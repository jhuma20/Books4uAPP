### Prerequisites

- Node.js (>= 16.x)

### Steps
1. **Clone the Repository:**
   ```sh
   git clone https://github.com/your-username/books4u.git
   cd books4u
   ```
2. **Backend Setup:**
   ```sh
   cd backend
   npm install
   npm start
   ```

   ```sh
   cd ../frontend
   npm install
   npm start
   ```
4. **Database Setup:**
   - Configure PostgreSQL and apply migrations.

5. **Run the Application:**
   Open `http://localhost:3000/` in your browser to explore Books4u!
